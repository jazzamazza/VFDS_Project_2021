**<br>HOW TO COMPILE VFDS:**

>NB Requires:
>
>- qmake
>- make
>- qt

Run cmd in terminal:
	
	% qmake -project
	% qmake
	% make

**HOW TO RUN VFDS:**

MAC and QMake:

	% ./vfds.app/Contents/MacOS/vfds

Linux and QMake:

	% ./vfds

DRIVER:
	
	make
	./drv <foldername>

UNIT TESTING:

	make utest
	./utest
